# export attack-log

Export war attack history.

## Options

| Name | Description |
|------|-------------|
| `clans` | Clan tags or aliases to filter clans. |
| `war_type` | Regular or friendly wars (defaults to Regular)<details><summary>View Options</summary>`Regular`, `CWL`, `Friendly`</details> |
| `limit` | Number of wars (Default: 25) |
