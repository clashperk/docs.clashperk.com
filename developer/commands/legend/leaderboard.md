# legend leaderboard

Shows legend leaderboard.

## Options

| Name | Description |
|------|-------------|
| `clans` | Clan tags or aliases to filter clans. |
| `limit` | Limit the number of results. |
| `season` | Retrieve data for the specified season.<details><summary>View Options</summary>`Oct 2025`, `Sep 2025`, `Aug 2025`, `Jul 2025`, `Jun 2025`, `May 2025`, `Apr 2025`, `Mar 2025`, `Feb 2025`, `Jan 2025`, `Dec 2024`, `Nov 2024`, `Oct 2024`, `Sep 2024`, `Aug 2024`, `Jul 2024`, `Jun 2024`, `May 2024`</details> |
| `enable_auto_updating` | Enable auto updating (every 30-60 mins)<details><summary>View Options</summary>`Legend Trophies`, `Builder Trophies`</details> |
