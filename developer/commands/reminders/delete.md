# reminders delete

Delete a reminder by ID (do /reminders list to get the ID)

## Options

| Name | Description | Type/Options |
|------|-------------|--------------|
| `type` | Type of the reminder. | `Clan Wars`, `Capital Raids`, `Clan Games` |
| `id` | Reminder ID (use /reminders list to get the ID) | `String` |
