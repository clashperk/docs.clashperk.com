# whitelist

Whitelist a role or user to use specific commands.

## Options

| Name           | Description                           |
| -------------- | ------------------------------------- |
| `user_or_role` | User or role to whitelist.            |
| `command`      | Command to whitelist.                 |
| `clear`        | Clear the whitelist.                  |
| `list`         | List all whitelisted users and roles. |
