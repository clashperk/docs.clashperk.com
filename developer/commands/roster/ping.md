# roster ping

Ping members in the roster

## Options

| Name | Description |
|------|-------------|
| `roster` | Select a roster to ping |
| `message` | Message for the members |
| `ping_option` | Select a ping option<br/>[`Unregistered (didn't signup, but in the clan)`, `Missing (opted-in, but not in the clan)`, `Everyone (all opted-in members)`] |
| `group` | Select a group to ping (overwrites ping_option) |
