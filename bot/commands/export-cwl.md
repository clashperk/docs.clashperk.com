# export cwl

Export CWL stats to Excel.

## Options

| Name          | Description                               |
| ------------- | ----------------------------------------- |
| `clans`       | Clan tags or aliases to filter clans.     |
| `season`      | Retrieve data since the specified season. |
| `wars`        | Number of wars (Default: 25)              |
| `lineup_only` | Export only the lineup.                   |
