# export last wars

Export participation history (last played wars)

## Options

| Name       | Description                               |
| ---------- | ----------------------------------------- |
| `clans`    | Clan tags or aliases to filter clans.     |
| `war_type` | Regular or CWL                            |
| `season`   | Retrieve data since the specified season. |
| `limit`    | Number of wars (Default: 25)              |
