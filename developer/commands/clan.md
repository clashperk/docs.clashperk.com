# clan

Shows comprehensive overview of a clan.

## Options

| Name | Description | Type/Options |
|------|-------------|--------------|
| `tag` | Clan tag or name or alias. | `String` |
| `user` | Clan by @user mention or ID. | `User` |
| `by_player_tag` | Get clan by a player tag. | `String` |
