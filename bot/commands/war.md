# war

Shows war summary and overview of a clan.

## Options

{% hint style="info" %}
Provide one of the options below to specify which clan or war to show.
{% endhint %}

| Name     | Description                  |
| -------- | ---------------------------- |
| `clan`   | Clan tag or name or alias.   |
| `user`   | Clan by @user mention or ID. |
| `war_id` | Search by war ID.            |
