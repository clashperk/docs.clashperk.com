# link delete

Deletes a player account/clan from a Discord account.

## Options

| Name | Description | Type/Options |
|------|-------------|--------------|
| `player_tag` | The player tag to unlink. | `String` |
| `clan_tag` | The clan tag to unlink. | `String` |
