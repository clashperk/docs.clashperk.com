# flag delete

Delete a player flag.

## Options

| Name | Description |
|------|-------------|
| `flag_type` | The type of the flag (ban or strike)<br/>[`Ban`, `Strike`] |
| `player` | The tag of a player to delete. |
| `flag_ref` | Flag reference of this player. |
| `clan` | Clan tag or name or alias. |
