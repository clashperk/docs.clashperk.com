# export missed

Export missed attack history.

## Options

| Name | Description |
|------|-------------|
| `clans` | Clan tags or aliases to filter clans. |
| `war_type` | Regular, CWL or Friendly Wars (defaults to Regular)<details><summary>View Options</summary>`Regular`, `CWL`, `Friendly`, `Regular and CWL`</details> |
| `season` | Retrieve data since the specified season.<details><summary>View Options</summary>`Since Nov 2025`, `Since Oct 2025`, `Since Sep 2025`, `Since Aug 2025`, `Since Jul 2025`, `Since Jun 2025`, `Since May 2025`, `Since Apr 2025`, `Since Mar 2025`, `Since Feb 2025`, `Since Jan 2025`, `Since Dec 2024`, `Since Nov 2024`, `Since Oct 2024`, `Since Sep 2024`, `Since Aug 2024`, `Since Jul 2024`, `Since Jun 2024`</details> |
| `limit` | Number of wars (Default: 25) |
