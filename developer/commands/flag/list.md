# flag list

List all player flags.

## Options

| Name | Description | Type/Options |
|------|-------------|--------------|
| `flag_type` | The type of the flag (ban or strike) | [`Ban`, `Strike`] |
| `player` | Show all flags against a player | `String` |
| `clans` | Clan tags or aliases to filter clans. | `String` |
