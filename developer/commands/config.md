# config

Configure server settings.

## Options

| Name | Description | Type/Options |
|------|-------------|--------------|
| `bot_manager_role` | Role that can manage the bot. | `Role` |
| `roster_manager_role` | Role that can manage the roster members. | `Role` |
| `flags_manager_role` | Role that can manage the flags. | `Role` |
| `links_manager_role` | Role that can manage the Discord links. | `Role` |
| `color_code` | Hex color code (e.g #ed4245) | `String` |
| `webhook_limit` | The maximum number of webhooks that can be created in a channel. | `Integer` |
