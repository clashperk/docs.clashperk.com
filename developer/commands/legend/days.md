# legend days

Shows per-day legend attacks for a player.

## Options

| Name | Description | Type/Options |
|------|-------------|--------------|
| `player` | Player by @user mention or ID. | `String` |
| `user` | Clan by @user mention or ID. | `User` |
| `day` | The league day. | `Number` |
