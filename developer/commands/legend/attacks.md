# legend attacks

Shows per-day legend attacks for a clan.

## Options

| Name | Description | Type/Options |
|------|-------------|--------------|
| `clans` | Clan tags or aliases to filter clans. | `String` |
| `user` | Player by @user mention or ID. | `User` |
| `day` | The league day. | `Number` |
