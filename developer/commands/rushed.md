# rushed

Rushed units and rushed % of a player or clan members.

## Options

| Name | Description | Type/Options |
|------|-------------|--------------|
| `player` | Player tag or name. | `String` |
| `user` | Player by @user mention or ID. | `User` |
| `clan` | Displays rushed units of a clan. | `String` |
