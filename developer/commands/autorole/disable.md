# autorole disable

Disable automatic role management for clan roles, town hall levels, or leagues.

## Options

| Name | Description |
|------|-------------|
| `type` | Type of roles to disable.<br/>[`Clan Roles`, `Town Hall Roles`, `Leagues Roles`, `Builder Hall Roles`, `Builder Leagues Roles`, `War Roles`, `End of Season Push Role`, `Family Leaders Role`, `Family Roles`, `Exclusive Family Role`, `Guest Role`, `Verified Role`, `Account Linked Role`] |
| `clans` | Clan tags or aliases to filter clans. |
