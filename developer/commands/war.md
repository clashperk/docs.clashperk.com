# war

Shows war summary and overview of a clan.

## Options

| Name | Description | Type/Options |
|------|-------------|--------------|
| `clan` | Clan tag or name or alias. | `String` |
| `user` | Clan by @user mention or ID. | `User` |
| `war_id` | Search by war ID. | `String` |
