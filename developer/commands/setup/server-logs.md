# setup server-logs

Setup automatic logs for the server.

## Options

| Name | Description |
|------|-------------|
| `log_type` | Select the log type to setup.<br/>[`Flag Alert Log`, `Roster Change Log`, `Maintenance Break Log`, `Welcome Log`] |
| `disable` | Whether to disable the log.<br/>[`Yes`, `No`] |
