## Options

| Name    | Description                                                    |
| ------- | -------------------------------------------------------------- |
| `code`  | Code to evaluate.                                              |
| `shard` | Whether to run the code on all shards or just the current one. |
| `depth` | Depth of the returned object.                                  |
