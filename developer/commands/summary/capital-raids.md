# summary capital-raids

Shows information about capital raids.

## Options

| Name | Description |
|------|-------------|
| `clans` | Clan tag or name or alias. |
| `week` | Retrieve data for the specified raid weekend.<br/>[`31 Oct, 2025`, `24 Oct, 2025`, `17 Oct, 2025`, `10 Oct, 2025`, `03 Oct, 2025`, `26 Sep, 2025`] |
