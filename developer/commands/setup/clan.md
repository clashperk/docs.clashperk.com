# setup clan

Manage clans on the server (link server, link channel, clan embed)

## Options

| Name | Description |
|------|-------------|
| `clan` | Clan tag or name or alias. |
| `category` | Category of the clan. (select from the menu or type your own) |
| `clan_channel` | Link the clan to a channel. |
| `unlink_clan_channel` | Unlink a channel from the clan. |
| `unlink_clan` | Unlink a clan from the server and remove all the features related to it. |
